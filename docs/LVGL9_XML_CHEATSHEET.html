Layout Systems & Widgets | LVGL 9.4

Layout & Position

Base Attributes

name str for lv_obj_find_by_name()
width size
height size
x size from left edge
y size from top edge
align enum position within parent
center, top_left, top_mid, top_right, bottom_*, left_mid, right_mid
Size values: px (100), % (50%), "content" (auto-size to content)

Flex Layout

Attributes

flex_flow enum direction of items
flex_grow int 0=fixed, 1+=fills space
flex_in_new_track bool wrap to next line
row, column, row_reverse, column_reverse, *_wrap

Style Properties

style_flex_main_place justify-content
style_flex_cross_place align-items
style_flex_track_place align-content (wrap)
start, center, end, space_between, space_around, space_evenly
⚠️ flex_align doesn't exist! Use style_flex_* instead

Grid Layout

style_grid_column_align
style_grid_row_align
style_grid_cell_column_pos col index
style_grid_cell_column_span cols wide
style_grid_cell_row_pos row index
style_grid_cell_row_span rows tall
align: start, end, center, stretch, space_*

lv_label

text string
long_mode enum text overflow
bind_text subject [fmt] reactive binding
long_mode: wrap, scroll, scroll_circular, dots, clip
bind_text: "subject_name '%.1f°C'"

Interactive Widgets

lv_button

Inherits all lv_obj attributes only

lv_checkbox / lv_switch

text string (checkbox only)
checked bool
bind_checked subject reactive
Switch is toggle without text label

lv_slider / lv_bar

value int [anim] current val
start_value int [anim] range mode
range min max "0 100"
bind_value subject reactive
mode enum
orientation enum
mode: normal, range, symmetrical | orientation: auto, horizontal, vertical

lv_image

src image
scale_x 256=100% 512=200%
scale_y 256=100% 128=50%
rotation 0.1° units 900=90°
inner_align enum within bounds
pivot x y rotation center
inner_align: center, stretch, tile, top_*, bottom_*
⚠️ zoom doesn't exist! Use scale_x/scale_y

lv_arc / lv_spinner

lv_arc

value int
range min max
bg_start_angle 0-360
bg_end_angle 0-360
rotation 0.1° units
mode enum
mode: normal, reverse, symmetrical

lv_spinner

Auto-rotating arc - no additional attributes

lv_dropdown / lv_roller

options string "\n" separated
selected int index (0-based)
visible_row_count (roller only)
dir (dropdown only)
dir: bottom, top, left, right
Options: "Option 1\nOption 2\nOption 3"

lv_keyboard / lv_textarea

lv_keyboard

mode enum
textarea target textarea ptr
mode: text_lower, text_upper, special, number

lv_textarea

text string
placeholder_text
max_length int
one_line bool
password_mode bool
accepted_chars filter input

Styles, Events, & Behavior | LVGL 9.4

Style: Size & Spacing

Padding

style_pad_all inside widget
style_pad_hor left+right
style_pad_ver top+bottom
style_pad_gap between flex items
style_pad_left/right
style_pad_top/bottom

Margin

style_margin_all outside widget
style_margin_hor
style_margin_ver

Size & Constraints

style_radius rounded corners
style_width/height
style_min_width/height prevent shrinking
style_max_width/height limit growth

Style: Background

style_bg_color 0xRRGGBB
style_bg_opa 0-255/% opacity
style_bg_grad_dir gradient direction
style_bg_grad_color end color
style_bg_image_src
style_bg_image_tiled repeat pattern
style_bg_image_recolor tint color
style_bg_image_recolor_opa tint strength
grad_dir: none, hor, ver

Style: Text

style_text_color
style_text_font font name
style_text_opa
style_text_align within label
style_text_letter_space kerning
style_text_line_space leading
style_text_decor
align: left, right, center, auto | decor: none, underline, strikethrough

Style: Border & Shadow

Border

style_border_color
style_border_width
style_border_opa
style_border_side which edges
side: none, top, bottom, left, right, full

Shadow

style_shadow_width blur amount
style_shadow_color
style_shadow_opa
style_shadow_offset_x horizontal shift
style_shadow_offset_y vertical shift

Style: Effects

style_opa 0-255 transparency
style_blend_mode compositing
style_transform_scale_x 256=100%
style_transform_scale_y 256=100%
style_transform_rotation 0.1° units
style_translate_x shift left/right
style_translate_y shift up/down
style_image_recolor tint color
style_image_recolor_opa tint strength
blend_mode: normal, additive, subtractive, multiply, difference
⚠️ style_image_recolor not style_img_recolor!

<style> Elements

Definition

<styles>
  <style name="btn"
         bg_color="0x2196f3"
         radius="8"
         pad_all="12"/>
</styles>

Application with Selectors

<lv_button>
  <style name="btn"/>
  <style name="btn_pressed"
         selector="pressed"/>
  <style bg_color="0xff0000"
         selector="checked"/>
</lv_button>
Selectors: default, pressed, focused, checked, disabled, etc.
Drop style_ prefix: "bg_color" not "style_bg_color"

Event Callbacks

XML Syntax (9.4+)

<lv_button>
  <event_cb trigger="clicked"
             callback="my_handler"/>
</lv_button>

Common Triggers

clicked
value_changed
pressed
released
ready keyboard done
cancel keyboard cancel
Register handler: lv_xml_register_event_cb("my_handler", fn)

Flags (Boolean)

Core Flags

hidden visibility
clickable interaction
checkable
floating ignore layout
ignore_layout
overflow_visible

Scroll Flags

scrollable
scroll_dir hor/ver/all
scroll_elastic bounce
scroll_momentum
scroll_chain to parent
scroll_on_focus
scroll_one snap

Advanced

event_bubble
snappable
click_focusable
flex_in_new_track
⚠️ Never use flag_ prefix! Just "hidden", not "flag_hidden"

States & Binding

State Attributes

checked checkboxes, toggle
focused input focus
disabled grayed out
pressed button press

Direct Bindings (attributes)

bind_text "subject 'format'"
bind_value "subject"
bind_checked "subject"

Conditional Bindings (child elements)

⚠️ Use child elements, NOT attributes!
bind_flag_if_eq/ne/gt/ge/lt/le
bind_state_if_eq/ne/gt/ge/lt/le
<lv_obj>
  <lv_obj-bind_flag_if_eq
      subject="step"
      flag="hidden"
      ref_value="2"/>
</lv_obj>
subject: data source | flag/state: which to set | ref_value: comparison value
